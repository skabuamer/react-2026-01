🚀 Eventics - Tailwind Event & Conference React JS Template

Eventics – is a Tailwind CSS React JS template designed for Conference, Event, Meetup, Seminar, Exhibition, Congress, Meeting, Business Conference, Event Management, Education Event, Meet Up websites.

  • Author: Tempticspro
  • Version: 1.0
React Tailwind Reusable Responsive

✨ Features

  • Fully Responsive Design
  • Modern React Hooks & Functional Components
  • Reusable Component Structure
  • Clean and Well Commented Code
  • Fast Performance (Vite Build)
  • Cross Browser Compatible

⚡ Requirements

  • Node.js (v16 or higher recommended)
  • npm or yarn
  • Modern Browser (Chrome, Edge, Firefox)

📦 Installation

npm install
npm run dev

⚙️ Production Build

npm run build

This generates a production-ready build folder.

📁 Folder Structure
